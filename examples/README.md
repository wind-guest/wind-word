# Examples

运行方式：

```bash
go run ./examples/basic
go run ./examples/images_tables
go run ./examples/template
go run ./examples/markdown
go run ./examples/math
```

生成的文件会写到 `examples/output/`。

说明：

- `basic`: 基础文档、标题、目录、列表、脚注、表格
- `images_tables`: 图片、浮动图片、单元格图片、表格操作
- `template`: 模板变量、条件、循环、图片占位符
- `markdown`: Markdown 和 Word 双向转换
- `math`: 直接 API 的行内公式和块级公式
